# Plateforme de jeux

Auteur : Camille

Date de début : 24/10/19

Contexte : Projet Java 2 Faire un plateau de jeu


# Description générale

La Plateforme permet de jouer à différents jeux.
1 Puissance 4
2 Morpion
3 Pendu
Les jeux se déroulent dans la console.
Les jeux Puissance 4 et Morpion se jouent soit en Joueur contre Joueur ou Joueur contre Ordinateur (Le Morpion supporte Ordinateur contre Ordinateur)
Elle est développé en Java.
